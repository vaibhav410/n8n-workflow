# 🚀 AI Job Automation System (n8n Workflow)

🔥 Built in Public | Open Source  

Stop applying for internships. Let them come to you.

This project is a fully automated job-hunting system built using **n8n + Google Gemini AI** that finds relevant internships/jobs and delivers them directly to your inbox.

---

## ⚡ Features

- 🔍 Automatically fetches latest jobs from LinkedIn  
- 🤖 AI filters jobs based on your skills  
- ✉️ Generates smart email / cover letter  
- 📊 Stores jobs in Google Sheets  
- 📬 Sends daily job alerts via Gmail  
- ⏰ Runs automatically every day (5 PM)  

---

## 🛠 Tech Stack

- n8n (Workflow Automation)  
- Google Gemini API (AI)  
- Gmail API (Email sending)  
- Google Sheets API (Tracking)  
- LinkedIn (Job Source)  

---

## 🚀 How It Works

1. Fetch jobs from LinkedIn  
2. Extract job data (title, company, link)  
3. Filter relevant jobs using AI  
4. Generate personalized content  
5. Store in Google Sheets  
6. Send email notification  

---

## 📥 How to Use

### Step 1: Import Workflow

- Open n8n  
- Click **Import from URL**  
- Paste this link:

https://raw.githubusercontent.com/vaibhav410/n8n-workflow/main/job%20Automation.json

---

### Step 2: Setup Credentials

You need to connect:

- ✅ Gmail account (for sending emails)  
- ✅ Google Sheets account  
- ✅ Google Gemini API key  

---

### Step 3: Customize Inputs

Edit these fields in workflow:

- Keywords (e.g. "Python Intern", "AI Intern")  
- Location  
- Job Type (Internship / Full-time)  

---

### Step 4: Activate Workflow

- Click **Activate**  
- It will run automatically daily  

---

## ⏰ Automation Schedule

- Runs every day at **5:00 PM**  
- Sends fresh job opportunities directly to your inbox
